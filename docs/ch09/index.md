# Chapter 9: Hypothesis Testing

## Overview

This chapter develops the framework for making decisions about population parameters based on sample data. We cover null and alternative hypotheses, test statistics, p-values, one-sample and two-sample tests, errors and power, and multiple testing corrections.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
