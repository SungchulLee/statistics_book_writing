# Chapter 2: Descriptive Statistics

## Overview

This chapter covers the tools for exploring, summarizing, and visualizing data. We examine histograms, ECDFs, measures of center and spread, and various visualization techniques for comparing distributions and groups.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
