# Chapter 1: Data Collection and Learning from Data

## Overview

This chapter explores how data is collected and how we learn from it. We contrast the classical approach (designed experiments, surveys, sampling) with the modern approach (learning from available data), and introduce the three major learning paradigms: supervised, unsupervised, and reinforcement learning.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
