# Chapter 4: Distributions and Dependence

## Overview

This chapter catalogs the most important probability distributions (discrete and continuous) and develops the machinery for working with multiple random variables: joint, marginal, and conditional distributions, covariance, and correlation.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
