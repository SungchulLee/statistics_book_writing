# Chapter 10: Chi-Square Tests

## Overview

This chapter covers chi-square tests for categorical data. We examine the chi-square distribution, then apply it to goodness-of-fit testing, tests of independence, and tests of homogeneity, along with practical considerations like expected cell counts and effect size.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
