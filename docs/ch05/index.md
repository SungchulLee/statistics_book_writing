# Chapter 5: Sampling Distributions

## Overview

This chapter bridges probability and inference. We show that statistics are random variables, introduce the four fundamental sampling distributions (Normal, t, Chi-Square, F), and derive the sampling distributions of common statistics.

## Learning Objectives

After completing this chapter, you should be able to understand and apply the key concepts covered in each section below.
